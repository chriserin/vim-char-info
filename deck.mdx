# How much information can we fit into a *character*?

---

# A musical note has *pitch*

Where the character is along the y axis (how many lines down) determines the pitch of the note

---

# A musical note has *timing*

Where the character is along the x axis (how many columns from the left) determines the timing of the note

---

# A musical note has *volume*

What letter the character is, a-zA-Z, will determine the volume (or _velocity_) of the note

# Combining Characters in UTF-8

Diacritical Marks such as <Mark> ̈</Mark> or <Mark> ̃</Mark> can be combined with the charater to include more information

<Char>A</Char> plus <CharMark> ̈</CharMark> equals <Char>Ä</Char>

or

<Char>\u0041</Char> plus <Char>\u0308</Char> equals <Char>\u0041\u0308</Char>

# Dialectics can represent eigth note or sixteenth note patterns

#### Eights

<MarkList>
<li><Char>ö</Char> - both notes</li>
<li><Char>ȯ</Char> - first note</li>
<li><Char>o͘</Char> - second note</li>
</MarkList>

#### Sixteenths

<MarkList>
<li><Char>ȍ̋</Char> - all four notes</li>
<li><Char>ȍ</Char> - first two notes</li>
<li><Char>ő</Char> - last two notes</li>
<li><Char>ò</Char> - first note</li>
<li><Char>ó</Char> - last note</li>
<li><Char>o̎</Char> - middle two notes</li>
<li><Char>o͆</Char> - first and last notes</li>
</MarkList>

# What else?

A dialectic could represent pitches for those rhythms

# What else?
