How much information can we fit into a character?

A musical note has pitch

Where the character is along the y axis (how many lines down) determines the pitch of the note

A musical note has timing

Where the character is along the x axis (how many columns from the left) determines the timing of the note

A musical note has volume

What letter the character is, a-zA-Z, will determine the volume (or velocity) of the note

Problem

Can a character represent more than one note?

If a column represents one beat, how do I represent eighth notes, sixteenth notes or triplets?

Combining Characters in UTF-8

Diacritical Marks such as ̈ or ̃ can be combined with the charater to include more information

A plus ̈ equals Ä

or

\u0041 plus \u0308 equals \u0041\u0308

Dialectics can represent eigth note or sixteenth note patterns

Eights

  • ö - both notes
  • ȯ - first note
  • o͘ - second note

Sixteenths

  • ȍ̋ - all four notes
  • ȍ - first two notes
  • ő - last two notes
  • ò - first note
  • ó - last note
  • o̎ - middle two notes
  • o͆ - first and last notes

What else?

A dialectic could represent pitches for those rhythms

̬ could combine to this Ä̬ and be up-down-up.

What else?

A dialectic could represent articulation

̻ could combine with this A̻ to be a snappier envelope.
